# SetupRadar KR Copy Editing Workbench

이 문서는 v0.2.9 기준 **Part 1** 작업대입니다. 공개 사이트 기능이 아니라, yulxwell이 한국어 문구를 검토하고 수정 지시를 남기기 위한 내부 편집 문서입니다.

## 0. 사용 방법

- 이 문서는 사이트 문구를 직접 고치기 위한 작업대입니다.
- "현재 문구"는 지금 사이트 또는 `src/content`에 있는 문구입니다.
- "운영자 수정안"에 yulxwell이 원하는 새 문구를 적습니다.
- 실제 사이트 반영은 나중에 Codex/Gemini가 수정안을 `editor*` 필드 또는 해당 copy/config 파일에 옮긴 뒤 `git push` -> Cloudflare Pages 배포로 진행합니다.
- 실시간 수정이 아닙니다.
- 제품/스위치/가이드 본문 데이터는 `ai*` 필드를 직접 덮어쓰기보다 `editor*` 필드에 반영하는 방식을 유지합니다.
- `id`, `value`, `href`, `iconName` 같은 시스템 연결 값은 문구가 아니므로 일반적으로 수정하지 않습니다.

권장 작성 방식:

```md
운영자 수정안:
> 초보자가 읽었을 때 바로 이해되는 문장으로 작성

메모:
> 너무 과장됨 / 더 캐주얼하게 / 브랜드 언급 줄이기 등
```
